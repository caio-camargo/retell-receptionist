# Retell Receptionist — Landing Page Prototype

A static landing page modeled on [reception.ai](https://reception.ai), positioning a Retell-branded AI receptionist product. Built as a fast prototype for review by Bing and Evie.

## Stack
Single `index.html` with Tailwind via CDN. No build step.

## Local preview
Open `index.html` in any browser, or:
```
npx serve .
```

## Deploy
GitHub Pages enabled on `main` → live at the Pages URL.

## Structure (mirrors reception.ai)
1. Nav
2. Hero — "Stop losing customers to voicemail"
3. Product intro with conversation mockup
4. 6 feature blocks
5. **Platform section** — the differentiator vs. reception.ai (architecture diagram, swap-any-provider, enterprise compliance)
6. FAQ
7. CTA
8. Footer

## Notes
- All CTAs link to `#` placeholders — wire to real signup once approved.
- Architecture diagram is `assets/architecture.jpg` (provided by Bing).
- Brand color `#FF6A2C` is a placeholder — swap for official Retell orange if/when needed.
